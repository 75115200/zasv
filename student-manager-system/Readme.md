##学生成绩管理系统##
这是大一第一学期c语言大作业的实现代码。总代码行数约1100Line。  
想想当时老师说要求1000Line代码时我和小伙伴们都惊呆了。然后我个实诚的人为了写完这一千行代码在学期结束前两个月就开始写。当时还是刚学C语言，对于模块化之类的东西完全不懂，也不懂得先设计好构架，总之就是上来就写代码。在我删了两次之后，终于写了一个能用的版本。想想当时删那两个辛辛苦苦写出来的雏形的时候是多么的心疼，r(st)q好不容易写了四五百行，最后因为代码凌乱，bug多多不得不删。  
待我辛辛苦苦敲完这一千行代码时才得知小伙伴们只写了几百行老师也收下了，甚至有拿学长学姐的冒充。。。。。 
一分耕耘一分收获，我倒不觉得自己亏了什么，要不是当时自己逼着自己写这么大代码量，哪来的现在动辄两千行井井有条！  
要说还真从这个大作业学了不少东西，第一次用链表，先搜百科、博客看看什么是链表，然后各种插入删除，甚至是链表排序的功能实现，自己对指针是真真的有个很好地了解、实践和掌握了。
